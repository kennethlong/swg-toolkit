## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| staging entry virtualPath → packPatch → filesystem | User-supplied virtual paths flow into file reads; path traversal could escape workspace |
| cfgActivator → client filesystem | cfgActivator writes into the client's cfg directory; wrong path = game config corruption |
| clientLocator → Windows registry | Registry is read-only probe; poisoned registry values could supply malicious install paths |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-08 | Tampering | packPatch virtualPath path traversal | mitigate | Reject any virtualPath containing '..' or starting with '/'|'\' before readFileSync; throw with explicit message |
| T-04-09 | Tampering | cfgActivator writing to wrong .cfg | mitigate | cfgPath argument is the toolkit-owned swgtoolkit.cfg path set by the user-confirmed client selection (D-04-11); validate path.isAbsolute and path ends in '.cfg'; never accept a path containing 'user.cfg' or 'options.cfg' |
