    2. SKTM inner chunk (NOT a FORM): read skeleton-name list required by this mesh (NUL-terminated strings). This is NOT parseSkeleton() — it is a flat name list. Store as sktmNames[]. Per synthesis delta #7: name distinctly as SktmNameList, not confused with the .skt root FORM.
    3. XFNM: transform/bone name table (NUL-terminated strings). Store as xfnmNames[].
    4. POSN: posCount × 3 float32 (global position pool). Cap at posCount × 12 bytes.
    5. TWHD: posCount × int32 (per-vertex influence count). Verify twhd array size == posCount.
    6. TWDT: flat (int32 transformIndex, float32 weight) pairs. Total count = sum of TWHD values. Cap at 4×posCount (reasonable upper bound).
    7. NORM (if present): normCount × 3 float32 (authored normals). Do NOT synthesize with computeVertexNormals — only populate if NORM chunk exists (synthesis anti-pattern).
    8. DOT3 (v0004 only): global tangent pool.
    9. PSDT* (per-shader groups): each contains NAME (shader path), PIDX (int32[] shader-local→POSN index), NIDX (int32[] → NORM), TXCI + TCSF/TCSD (UV sets), PRIM -> INFO + ITL (triangle index list: 3×int32 shader-local indices).
    10. For each PSDT: call deIndex(POSN pool, NORM pool, PIDX, NIDX, UVs, PRIM) -> DeIndexedBuffers.
