| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-13 | Tampering | palette index OOB | mitigate | Before PaletteArgb lookup: assert 0 <= index < entryCount; clamp to entryCount-1; UI swatch strip is bounded by entryCount |

| T-02-15 | Tampering | DDS mip offset OOB | mitigate | parseDds (02-01) validates sum of mip byteLengths <= (size - 128); decodeDxt also bounds-checks block reads against the slice length before indexing |
