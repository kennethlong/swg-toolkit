    parseAnimation(const swg_core::iff::IffNode& root) -> AnimationResult.
    Root tag discriminator: CKAT → CKAT-0001 path; KFAT + version read → 0003 path or 0002 graceful-decline path.

    KFAT-0002 path: read version INFO. If version == 2 (0002): return AnimationResult { variant:KFAT_0002_UNSUPPORTED, fps:0, frameCount:0, joints:{}, roundTrip }. Do NOT attempt parsing the Euler channels.

    KFAT-0003 path: FORM KFAT → FORM 0003.
    - INFO chunk: fps (float32), and 6 int32 counts (transformCount, staticRotationCount, animatedRotationCount, staticTranslationCount, animatedTranslationChannelCount, totalRotationFrames per KFAT-0003 field order). Cap transformCount at 2048.
    - FORM XFRM → XFIN × transformCount: each XFIN is a NUL-terminated name string. This maps joint index → name.
    - [FORM AROT] → [QCHN × animatedRotationCount]: each QCHN = { frameCount int32 + frameData: int32[] frame indices + float32[4][] quaternion data (w,x,y,z order per synthesis §1.6) }. Store per-joint quaternion keyframes as struct { frames[], quats[] (each (w,x,y,z) tuple) }.
    - [SROT] → static rotations: one quaternion (w,x,y,z) per static-rotation joint.
    - [FORM ATRN] → [CHNL × 3 axes]: animated translation channels (one per axis X/Y/Z, gated by translation_mask bits). Each CHNL = { frameCount + frame[] + value[] float32 }.
    - [STRN] → static translations: one Vector3 per static-translation joint.
    - Build keyframe IR: for each joint (transformCount), collect rotation keyframes (from AROT if animated, SROT if static, identity if absent) and translation keyframes (from ATRN X/Y/Z channels, STRN if static, zero if absent).
    - Pack IR into a flat ArrayBuffer: for each joint, frame-aligned struct: { quat w,x,y,z float32; tx,ty,tz float32 }. frameCount is maxKeyframeIndex+1 from the AROT/ATRN key lists.
    Oracle: KeyframeSkeletalAnimation.cpp.

    CKAT-0001 path: FORM CKAT → FORM 0001.
    - INFO chunk: fps (float32), and counts (int16): transformCount, staticRotationCount, animatedRotationCount, staticTranslationCount, animatedTranslationChannelCount. Cap transformCount at 2048.
    - FORM XFRM → XFIN names (same as KFAT).

The w CLAMP (max(0,…)) MUST be present — adopt it per synthesis §5 open item. expandElevenBit: lower 10 bits = valueMask, bit 10 = signBit; expandTenBit: lower 9 bits = valueMask, bit 9 = signBit.     Oracle: CompressedKeyframeAnimation.cpp + CompressedQuaternion.cpp:82-100,370-419.

    Security: transformCount/joint-count cap at 2048; frameCount per channel cap at 100,000; packed uint32 decode has no array OOB (it's arithmetic only); string names cap at 256 chars.
